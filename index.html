디자인과 개발능력을 겸비한
웹 퍼블리셔가 되기로 한 이유

2022년 4월 국민건강보험공단에서 주최한 UX디자인 전국공모전에서 2등(우수상)을 수상하면서
프로젝트 개발과정을 학습한다면 내가 디자인한 화면의 동적인 부분까지 작업할 수 있다는 생각이 들었습니다.

이를 위해 풀스택 개발자 취업캠프에 참여하여 HTML, CSS, JavaScript, jQuery, React, 를 배웠고,
Backend 기술까지 학습하여 디자인과 백엔드 개발자 사이에서의 원활한 커뮤니케이션을 할 수 있는 웹 퍼블리셔로 성장하고 있습니다.

작업하는 시간이 너무나도 즐거우며, 앞으로도 역량을 키워 더 나은 퍼블리셔로 성장하기 위해 노력하겠습니다.

평생건강정보조회서비스
APP아이디어 공모전 (우수상 수상)

건강보험공단에서 주최한 평생건강정보조회서비스APP 아이디어 공모전은
기존의 건강보험공단 앱에 추가될 예정인 "평생건강정보조회서비스"메뉴의
페이지를 디자인하는 공모전입니다.

"평생건강정보조회서비스"는 사용자의 건강정보를 생애주기별로
분류하여 한눈에 볼 수 있도록 합니다.

공모전 소개
- 목표 : 앱 내에 흩어져 있는 건강정보를 생애주기별로 분류하여

- 기대효과 : 자신의 건강정보를 헤매지 않고 경로가 한눈에 보일 수 있도록 하는 것

Check Points

    - 연령대에 상관없이 쉬운 정보파악
3. 사용자층이 "대한민국 국민 누구나"이기 때문에 디지털 약자여도 쉽게 이용할 수 있도록
    최대한 간결하고 인지하기 쉽게 디자인 하였으며
    찾으려는 정보들을 잘 찾을 수 있도록 설계함
4. 전체적인 계획을 날짜에 맞춰 세우고 단계별로 진행하여 지체됨 없이 프로젝트를 수행함
